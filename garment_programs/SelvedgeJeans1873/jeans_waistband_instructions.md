# Drafting the Waistband

Source: https://masterclasses.historical-tailoring.com/lesson/drafting-the-fly-and-waistband/

The waistband is a simple rectangular strip that can be drafted directly on the
cloth, though drafting on paper first is recommended for a first attempt.

## Steps

### 1. Draw the length

Draw a horizontal line equal to the **waist measurement** plus **3" to 4" on
each side**. The extra length allows for turning under the ends and provides
room for error if the waist measurement is slightly off.

### 2. Set the width

Total strip width: **3 3/4"**, broken down as follows (bottom to top):

| Section | Width |
|---------|-------|
| Selvedge edge (bottom inside) | 3/8" |
| Inside of waistband | 1 1/2" |
| **FOLD** | |
| Outside of waistband | 1 1/2" |
| Seam allowance (top) | 3/8" |

The fold line sits between the inside and outside sections.

## Cutting notes

- One long edge should be placed along the **selvedge** of the denim when
  cutting.
- If not using selvedge in the waistband (e.g., to save fabric), add another
  **1/2"** for safety on that edge.
